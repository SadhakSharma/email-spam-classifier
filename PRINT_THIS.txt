□ Download Python from python.org (CHECK "Add to PATH" during install!)
□ Create folder: email-spam-classifier
□ Open Command Prompt/Terminal IN THAT FOLDER
□ Type: python --version (should show Python 3.x.x)
□ Type: python -m venv venv
□ Type: venv\Scripts\activate (Windows) OR source venv/bin/activate (Mac)
□ Type: pip install flask scikit-learn pandas numpy
✓ Done! You'll see "(venv)" in your terminal

"python not found" 
→ Reinstall Python, CHECK "Add to PATH"

"ModuleNotFoundError"
→ Make sure (venv) shows in terminal
→ Run: pip install flask scikit-learn pandas numpy

"model.pkl not found"
→ You forgot to run python train_model.py
→ Run it first!

"Can't connect to localhost:5000"
→ Make sure python app.py is still running (don't close the window)
→ Try http://127.0.0.1:5000 instead

"Can't push to GitHub"
→ Make sure you're in the cloned folder (cd email-spam-classifier)
→ Make sure Git is installed (git --version)
→ Use HTTPS link when cloning (not SSH)

"Replit won't work"
→ In Replit console, run: python train_model.py
→ Wait for it to finish
→ Then click Run
